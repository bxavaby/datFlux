ENTROPY COLLECTION METHODOLOGY

datFlux creates background system load through various noise generation methods, collecting and hashing unpredictable system variations to create high-entropy passwords. The multi-level entropy quality gauge provides real-time feedback on randomness extraction.

INSTALLATION
03
QUICK INSTALL SCRIPT 
# clone the repo
git clone https://github.com/bxavaby/datFlux.git
cd datFlux

# run installer
./install.sh
MANUAL BUILD
# clone the repo
git clone https://github.com/bxavaby/datFlux.git
cd datFlux

# build the binary
go build -o datflux ./cmd/datflux

# run
./datflux
GO INSTALL
go install github.com/bxavaby/datFlux/cmd/datflux@latest
USAGE COMMANDS
datflux
Launch interactive TUI mode
datflux now
Generate password instantly (Standard Mode)
datflux now --paranoia
Generate high-security password (Paranoia Mode)
